## Overview

This folder contains a **full Python reproduction pipeline** for the Ngo et al. (2020) eLife study, which demonstrates that:

1. Sleep spindle power increases concurrently in hippocampus (HIPP) and neocortex (NC) time-locked to hippocampal ripple events.
2. Hippocampal-neocortical coherence in the spindle band (~12-16 Hz) is elevated around ripples.
3. Directionality analyses (Partial Directed Coherence, PDC) show NC drives HIPP (~250 ms before ripple peak).
4. These effects are specifically enhanced during **long-duration** compared to **short-duration** hippocampal ripples.

### Key Signals Analyzed
- **NC**: Scalp electrode Cz (neocortex, integrating fronto-parietal networks)
- **HIPP**: Intracranial EEG from posterior hippocampus
- **Recording**: Whole-night NREM sleep, 14 pre-surgical epilepsy patients

## Step-by-Step Reproduction Guide

### Step 0: Prerequisites

- Python 3.9+
- ~5 GB disk space for raw data + processed outputs
- Internet connection for data download

### Step 1: Set Up Python Environment

**Option A — Conda (recommended):**
```bash
conda env create -f environment.yml
conda activate ngo2020-repro
```

**Option B — pip virtualenv:**
```bash
python -m venv venv
# Linux/macOS:
source venv/bin/activate
# Windows:
venv\Scripts\activate

pip install -r requirements.txt
```

### Step 2: Download the Dataset

The raw EEG data (NC + HIPP signals for 14 subjects) is publicly available on OSF:

**DOI:** https://osf.io/3hpvr/

**Automatic download (recommended):**
```bash
python scripts/01_download_data.py
```
This downloads all subject `.mat` files (~14 subjects) into `data/raw/`.

**Manual download:**
1. Go to https://osf.io/3hpvr/
2. Click "Download as zip" or download individual subject files
3. Extract `.mat` files into `data/raw/` following the structure:
```
data/raw/
├── sub-01/
│   ├── sub-01_NC.mat      # Neocortex EEG (Cz)
│   └── sub-01_HIPP.mat    # Hippocampal iEEG
├── sub-02/
│   └── ...
...
└── sub-14/
```

**Expected `.mat` file variables (from original OSF data):**
- `data`: EEG signal array (channels x samples)
- `srate`: Sampling rate (typically 1000 Hz)
- `nrem_periods`: NREM sleep epoch indices

> **Note:** If the OSF file structure differs, inspect with:
> ```python
> import scipy.io
> mat = scipy.io.loadmat('data/raw/sub-01/sub-01_NC.mat')
> print(mat.keys())
> ```

### Step 3: Preprocess the Data

```bash
python scripts/02_preprocess.py
```

This script:
- Loads raw `.mat` files for each subject
- Extracts **artifact-free NREM sleep** segments
- Applies basic bandpass filtering (0.5–200 Hz)
- Saves cleaned NC and HIPP signals to `data/processed/`

### Step 4: Detect Spindles and Ripples

```bash
python scripts/03_detect_spindles_ripples.py
```

This implements the exact detection algorithm from Staresina et al. (2015) / Ngo et al. (2020):

**Spindle Detection (NC and HIPP):**
- Bandpass filter: 12–16 Hz
- Compute RMS with 200 ms window + smoothing
- Threshold: mean + 2.5 SD (detection) and mean + 9 SD (upper limit)
- Save spindle onset/peak/offset times

**Ripple Detection (HIPP only):**
- Bandpass filter: 80–100 Hz
- Compute RMS with 20 ms window
- Threshold: mean + 5 SD
- Reject events < 38 ms (< 3 cycles at 80 Hz) or > 500 ms
- Require minimum 3 cycles in raw signal
- Segment into 2 s epochs centered on positive peak

Outputs saved to `data/processed/detections/`

### Step 5: Time-Frequency Representation (TFR) Analysis

```bash
python scripts/04_tfr_analysis.py
```

Reproduces **Figure 2** of the paper:
- Compute TFR (Morlet wavelets, 4–30 Hz) for NC and HIPP signals
- Time-lock to individual ripple events (±1 s window)
- Normalize against matched ripple-free control events
- Compute group-level average and identify spindle-band (12–16 Hz) cluster

### Step 6: Coherence and Partial Directed Coherence (PDC) Analysis

```bash
python scripts/05_coherence_pdc.py
```

Reproduces **Figure 3** of the paper:
- Compute ripple-locked NC-HIPP coherence in the spindle band
- Compute **Partial Directed Coherence (PDC)** to determine directionality
  - NC → HIPP: neocortex driving hippocampus
  - HIPP → NC: hippocampus driving neocortex
- Expected result: NC → HIPP influence emerges ~250 ms before ripple peak

### Step 7: Long vs. Short Ripple Duration Analysis

```bash
python scripts/06_long_vs_short_ripples.py
```

Reproduces **Figure 4** of the paper:
- Split ripples into **long-duration** vs **short-duration** groups (median split)
- Compare NC spindle power between groups
- Expected: NC spindle power significantly higher during long-duration ripples
  (t-test, one-sided, as in Fernández-Ruiz et al., 2019)
- Compare NC-HIPP coherence for long vs. short ripples

### Step 8: Generate All Figures

```bash
python scripts/07_plot_figures.py
```

Generates publication-quality figures saved to `results/figures/`:
- `fig1_study_design.png` — Schematic + example data
- `fig2_tfr_ripple_locked.png` — NC and HIPP TFR around ripples
- `fig3_coherence_pdc.png` — Spindle-band coherence + PDC directionality
- `fig4_long_vs_short_ripples.png` — Duration-dependent effects

### Step 9 (Optional): Interactive Notebook

For exploratory analysis and step-by-step visualization:
```bash

```

## Troubleshooting

**Q: OSF download fails.**
A: Download manually from https://osf.io/3hpvr/ and place `.mat` files in `data/raw/`.

**Q: `scipy.io.loadmat` throws an error on large files.**
A: Use `mat73` package: `pip install mat73` and replace `scipy.io.loadmat` with `mat73.loadmat`.

**Q: Results differ from paper.**
A: Check:
1. Sampling rate matches (1000 Hz expected)
2. NREM scoring files loaded correctly
3. Artifact rejection thresholds in `config/config.yaml`
4. Compare against original MATLAB code at https://github.com/episodicmemorylab/Ngo_et_al_eLife2020

**Q: PDC computation is very slow.**
A: Reduce the number of bootstrap iterations in `config/config.yaml` (`pdc_n_bootstrap: 200` instead of 1000).
